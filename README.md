
# Test Form App

Form validation app


## Tech Stack

**Client:** ReactJS, MobX, SCSS, Axios, ReactRouterDom, classnames.

**API:** https://countriesnow.space


## Run Locally

Clone the project

```bash
  git clone https://github.com/sasaBej/winter-bootcamp_v2.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
